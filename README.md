# 10016-02-DevCon 1-Team 05
## **LedFoot**
### Version 1.0

**Controls :**
Move the mouse left and right OR swipe across your trackpad or touchscreen.
A head on collision will stop the car, avoid crashing.
You can run over obstacles.
